# GTA-SA-TXD-Editor


Arsitektur Editor Biner RenderWare Performa Tinggi dengan Proteksi Corrupt & Auto-Replace Duplikat untuk modding tekstur GTA San Andreas langsung di browser.
https://img.shields.io/badge/version-4.0-orange
https://img.shields.io/badge/license-MIT-blue
https://img.shields.io/badge/platform-Web-brightgreen
https://img.shields.io/badge/support-TXD%20%7C%20PNG%20%7C%20DXT1%2F3%2F5-purple




| Kategori                        | Deskripsi                                                                |
| ------------------------------- | ------------------------------------------------------------------------ |
| **🗂️ Parser Biner RenderWare** | Parsing file `.txd` asli GTA SA langsung di browser tanpa backend        |
| **🖼️ Decoder Multi-Format**    | Dukungan DXT1, DXT3, DXT5, RGBA 32-bit, RGB 24-bit, dan PAL8 (256 warna) |
| **🎨 Editor Palet 8-bit**       | Edit warna palet biner PAL8 secara real-time dengan color picker         |
| **⚡ Auto-Replace Duplikat**     | Deteksi otomatis nama tekstur duplikat dengan opsi timpa/abaikan         |
| **🔧 Batch Renamer**            | Ganti nama massal dengan prefix, find & replace                          |
| **📐 Optimasi Resolusi Massal** | Downscale/upscale batch untuk porting ke GTA SA Android/Mobile           |
| **🔍 Hex Inspector**            | Hex dump viewer untuk analisis struktur biner (max 1024 byte)            |
| **📊 Mipmap Generator**         | Auto-generate tingkatan mipmap dinamis via Canvas API                    |
| **💾 Ekspor Konfigurasi**       | Pengaturan alpha channel, platform target (PC/Android), rebuild mipmaps  |
| **🛡️ Proteksi Corrupt**        | Validasi format, force power-of-two, stabilizer resolusi                 |
